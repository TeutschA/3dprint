# Summary

This is a more complex and larger part based on the same basic geometry as Lumen2 and Lumen4. I added a base because there wasn't enough contact area between the part and the print bed, and it broke loose at about 50% complete. This part printed in 26 1/4 hours @ 0.200 mm layer height.
